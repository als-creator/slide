# Слайдер с выделением активного элемента

Простой JavaScript‑скрипт для создания слайдера с возможностью выделения активного элемента. При клике на слайд он получает класс `active`, а у остальных этот класс удаляется.


## Функционал

* При клике на любой слайд он становится активным (получает класс `active`).
* Автоматическое снятие класса `active` с ранее активного слайда.
* Простой и лёгкий код без зависимостей.
* Легко кастомизируется через CSS.
